JDBC - ReDBox Dataset Harvester Documentation

This documentation describes the JDBC to ReDBox Dataset Harvester Client, an implementation of Harvester Client that transfers data from JDBC-accessible databases into ReDBox. If you want to integrate your institution-specific database into ReDBox, you have come to the right place.
